Online Quiz

Hello! Welcome! In this online quiz, pay attention to the following rules:

  • The quiz consists of 10 questions in total.
  • There is only one correct answer for each question.
  • You can change the answer to the question you have answered, but you cannot change it after the time has expired.
  • Answer options will not be able to be clicked in the first 10 seconds and will be clickable after the 10th second.
  • You have 30 seconds for each question. Use your time carefully.
  • You cannot go back to past questions.
  • At the end of the quiz, your answers will be shown as a table.

Before starting the quiz, create a calm environment for yourself.

Select the user id given to you from the list, then you can start the quiz.

We wish you luck!
